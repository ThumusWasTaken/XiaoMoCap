WebUI Instructions:

Activate WiFi: Press the Xiao sensor button.
(LED flashes quickly, then turns OFF)
Connect: Join TOL-xxx WiFi network.
Configure:
Use the UI to configure rate and duration and click “Send Config”, or
Long press the sensor button (< 4 sec) to use default config.
(LED ON)
Start Recording: Press the sensor button.
(LED slow flash — WiFi off, data recording)
Stop Recording:
Wait for configured duration, or
Long press (< 4 sec) to stop early.
(LED quick flash — data ready)
Upload Data:
Press button to re-enable WiFi.
Reconnect to TOL-xxx WiFi and click “Connect” on UI.
(LED ON — data is sending)
Download:
Click “Download CSV” to save data early, it will automatically download when complete.
(LED OFF — finished)
